What Is It?

The Neurokernel Project aims to build an open software platform for the emulation of the entire brain of the fruit fly Drosophila melanogaster on multiple Graphics Processing Units (GPUs).

    How Can I Try It?

    Interested? Great!

    • Clone the source from GitHub.
    • Check out the project documentation and Wiki.
    • Take a look at the demos.

    Got a question or encountered problems?

    • Start a discussion on the project forum or mailing list.
    • Send in bug reports to the project issue tracker.

    Have a new feature to propose and add?

    • Propose a new RFC on the project forum or mailing list.
    • Send in code via a pull request.
